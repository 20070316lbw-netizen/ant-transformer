# Ant-Transformer 研究价值评估结论（2026-03-28）

## 结论（是否有研究价值）
有研究价值，但当前证据链不足以支持“结构性创新有效”的强结论；目前最多支持“存在可研究信号，尚未完成因果归因”。

## 是否值得继续深入
值得继续，但应从“工程可跑”转为“统计可证伪”的研究范式；在完成严格显著性与机制验证前，不建议沉淀为方法论结论。

## 下一步最优实验方向（最多3条）
1. **Gate 机制可解释性与塌缩检验**：做门控分布、层使用率、时间稳定性和置乱对照（gate randomization）实验，验证 gate 是否学习到稳定结构而非退化到常数策略。
2. **剪枝有效性的统计检验**：在统一数据切分与统一训练预算下做多随机种子重复实验，报告 IC/RankIC/Sharpe 的置信区间、配对检验与效应量，判断 Layer0 / Layer0+Layer2 / Full+Gate 差异是否显著。
3. **与 LightGBM 的差距归因实验**：做特征同源、标签同口径、成本同假设的公平基线；通过“表示学习收益 vs 过参数化惩罚”分解实验，明确深度结构在哪些市场状态/股票子集上能超越树模型。
